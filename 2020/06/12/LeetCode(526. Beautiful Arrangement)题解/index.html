发表于 2020-06-12
Medium，[526. 优美的排列](https://leetcode-cn.com/problems/beautiful-arrangement/)。我们将整数 x 的 权重 定义为按照下述规则将 x 变成 1 所需要的步假设有从 1 到 N 的 N 个整数，如果从这 N 个数字中成功构造出一个数组，使得数组的第 i 位 (1 <= i <= N) 满足如下两个条件中的一个，我们就称这个数组为一个优美的排列。条件：第 i 位的数字能被 i 整除。i 能被第 i 位上的数字整除。现在给定一个整数 N，请问可以构造多少个优美的排列？

LeetCode(\526. Beautiful Arrangement)题解

Suppose you have N integers from 1 to N. We define a beautiful arrangement as an array that is constructed by these N numbers successfully if one of the following is true for the ith position (1 <= i <= N) in this array:

  1. The number at the ith position is divisible by i.
  2. i is divisible by the number at the ith position.

Now given N, how many beautiful arrangements can you construct?

Note:

  1. N is a positive integer and will not exceed 15.

解题思路

Medium

这个题只是勇士回溯法就可以了，用一个包含N个数的数组，然后从0位进入递归，在第i层递归的时候。然后依次将后面的数换到第i位，当然是前提满足可以整除。如果最后能进入到第N层，说明这个序列是优美排列。返回1即可。
