LeetCode(\1497. Check If Array Pairs Are Divisible by k)题解

We want to divide the array into exactly n / 2 pairs such that the sum of each pair is divisible by k.

解题思路

Medium

这个题可以使用Greedy的办法直接统计。计算每个数基于K的余数，然后遍历所有1到k-1之间的数，如果刚好可以配对，同时0的个数为偶数，那么说明数组是可以分为两两组合配对的。反之则不可以。同时也可以建立了余数对应的Map之后，遍历所有的元素，然后看对应的需要的余数在不在Map内，在则将其-1，否则就说明不能配对。
