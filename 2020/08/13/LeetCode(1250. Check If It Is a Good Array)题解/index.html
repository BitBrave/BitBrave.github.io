LeetCode(\1250. Check If It Is a Good Array)题解

Hard

根据裴蜀定理：果两个数互斥，则存在整数 x,yx,y，使得 ax+by=1。推到多个数也成立。

证明：https://leetcode-cn.com/problems/check-if-it-is-a-good-array/solution/python3shu-xue-fang-fa-shi-jian-onlogm-by-gorilla/

一个补充：
1.若两个整数gcd为1，那么再添加任何整数到该集合中，其gcd还是1。
所以只要遍历一遍数组，如果gcd为1，必然存在两个互质的数。
