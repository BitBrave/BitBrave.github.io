LeetCode(\1551. Minimum Operations to Make Array Equal)题解

You have an array arr of length n where arr[i] = (2 * i) + 1 for all valid values of i (i.e. 0 <= i < n).

In one operation, you can select two indices x and y where 0 <= x, y < n and subtract 1 from arr[x] and add 1 to arr[y] (i.e. perform arr[x] -=1and arr[y] += 1). The goal is to make all the elements of the array equal. It is guaranteed that all the elements of the array can be made equal using some operations.

Given an integer n, the length of the array. Return the minimum number of operations needed to make all the elements of arr equal.

解题思路

Medium

本题直接分析即可。

返回一样的肯定是先计算所有数的平均数，然后将对应的配对数分别+1-1变成平均数。这个数列就是1,3,5,7,….,2n-1。这里可以分析中位数，就是平均数也就是n。因此，可以直接从1逐步将小于n的数变为n，那么对应的大于n的数肯定也全部变为n了。这样不用太多分析。

但是，如果仔细分析的话，会发现需要增加的其实是一个等差数列。当n为奇数时，最终结果为(n2-1)/4，当n为偶数是，结果为(n2)/4，而利用c++中int类型。我们可以直接返回(n2)/4。
