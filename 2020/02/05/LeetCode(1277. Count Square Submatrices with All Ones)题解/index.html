LeetCode(\1277. Count Square Submatrices with All Ones)题解

Given a m * n matrix of ones and zeros, return how many square submatrices have all ones.

Constraints:

  • 1 <= arr.length <= 300
  • 1 <= arr[0].length <= 300
  • 0 <= arr[i][j] <= 1

解题思路

给出一个矩阵A，将其中所有由1组成的平方子矩阵找出来，统计其数量。

这个题可以使用DP解决，设定为和A一样大小。然后DP[i,j]表示A中0i-1和0j-1之间矩阵内的所有全1子矩阵之和。那么状态转移方程就是：

  • DP[i+1,j+1] = DP[i,j+1] + DP[i+1,j] - DP[i, j] + 包括A[i,j]在内的向上向左所有全1子矩阵。

最后只需要返回DP[row, col]即可。

时空复杂度

代码如下，时间上，计算矩阵的全1子矩阵只需要O(n2)，最后构建状态转移方程时，需要O(n3)，空间上，需要O(n2)，但是状态转移方程观察只需要记录前一行和当前行左边的值，因此这个矩阵可以只需要O(n)的空间，但是记录子矩阵和的矩阵需要O(n2)，所以总的还是O(n2)。
