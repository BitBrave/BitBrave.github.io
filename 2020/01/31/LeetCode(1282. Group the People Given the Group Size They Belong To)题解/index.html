LeetCode(\1282. Group the People Given the Group Size They Belong To)题解

There are n people whose IDs go from 0 to n - 1 and each person belongs exactly to one group. Given the array groupSizes of length n telling the group size each person belongs to, return the groups there are and the people’s IDs each group includes.

You can return any solution in any order and the same applies for IDs. Also, it is guaranteed that there exists at least one solution.

Example 2:

Constraints:

  • groupSizes.length == n
  • 1 <= n <= 500
  • 1 <= groupSizes[i] <= n

解题思路

N个人有0到N-1的ID，每个人有对应的所属组，在数组中，i位置的元素表示ID为i的人所属的组的成员个数。现在需要根据组大小将其按照组分开，保证一定有解。

这个题稍稍思考一下就可以得出解法，首先记录下每个组员的ID和对应的组数，可以使用c++中的pair实现，然后将其按照组的大小排序，那么具有相同组数的一定是挨着的，然后依次取就可以。要注意的是可能有多个组大小是一样的，因此选择的时候要判断一下。

时空复杂度

代码如下，时间复杂度O(nlogn), 空间复杂度O(n). 这里其实还有一种O(n)时间复杂度的方法，就是按照组大小将其放入不同的set中，然后再将其分开，只是有点麻烦。
